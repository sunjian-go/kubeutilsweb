# 多集群页面操作pod工具前端
支持页面上传文件到pod内部，从pod内部下载文件到本地，终端、日志功能。支持多集群

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
